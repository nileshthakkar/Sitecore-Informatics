Sitecore Informatics

Sitecore Informatics is a kind of dashboard which will show you all your Sitecore Instance details at one place.

View project on GitHub

Sitecore Informatics

This tool will help you to track your Sitecore Website information such as:

  • Total number of Sites in your Sitecore instance
  • Total number of Sublayouts in your Sitecore instance along with referrer pages and caching parameters
  • Total number of Templates in your Sitecore instance along with referrer pages
  • Total number of Content Items in your Sitecore instance
  • Total number of Media Items in your Sitecore instance
  • Total number of Users in your Sitecore instance

How to download and use it?

It's very simple to download and install Sitecore Informatics tool, you simply need to download "Sitecore Informatics-V1" package from github package folder and install it in Sitecore.

Configuration:

After installing Sitecore Informatics, please make sure you have configured correct settings as suggested below: Go to Website\App_Config\Include\SitecoreInformatics.config and configure your settings: <!-- Specify sublayout root folder path in below setting. For example: /sitecore/layout/Sublayouts/[Site Sublayout Root FolderName]
If you keep this setting as blank, then Sitecore Informatics tool will consider default sublayout folder path as: /sitecore/layout/Sublayouts/[Your Sitecore Instance HomeRoot ItemName] --> 

    <!-- Specify template root folder path in below setting.
         For example: /sitecore/templates/User Defined/[Site Template Root FolderName]  
         If you keep this setting as blank, then Sitecore Informatics tool will consider default template folder path as:
         /sitecore/templates/User Defined/[Your Sitecore Instance HomeRoot ItemName]
    -->
    <setting name="TemplateFolderPath" value="" />

    <!-- Specify media root folder path in below setting.
         For example: /sitecore/media library/[Site Media Root FolderName]  
         If you keep this setting as blank, then Sitecore Informatics tool will consider default media folder path as:
         /sitecore/media library/[Your Sitecore Instance HomeRoot ItemName]
    -->
    <setting name="MediaFolderPath" value="" />

How to move it on CDN Server?

In case, if you want to use this tool on CDN Server then you should copy below files to server:

1) bin\SitecoreInformatics.DLL

2) bin\ICSharpCode.SharpZipLib.DLL

3) App_Config\Include\SitecoreInformatics.config

4) sitecore modules\SI folder
